# FedEx (fedex)

FedEx is a logistics company that provides shipping and delivery services worldwide. They offer a range of solutions for individuals and businesses, including express shipping, freight services, and e-commerce fulfillment. The FedEx Developer Portal publishes a suite of REST APIs covering tracking, shipping, rating, address validation, pickup, location lookup, authorization, and post-shipment visibility via webhooks.

## APIs

### FedEx Track API
Up-to-the-minute package and shipment status, scan events, delivery details, and proof of delivery using tracking numbers, reference numbers, or TCN.
- [Documentation](https://developer.fedex.com/api/en-us/catalog/track/v1/docs.html)

### FedEx Ship API
Create domestic and international shipments, generate shipping labels, validate addresses, schedule pickups, and manage end-to-end shipment workflows.
- [Documentation](https://developer.fedex.com/api/en-us/catalog/ship/v1/docs.html)

### FedEx Rate API
Rate quotes and transit times for FedEx Express, Ground, Freight, and SmartPost services.
- [Documentation](https://developer.fedex.com/api/en-us/catalog/rate/v1/docs.html)

### FedEx Address Validation API
Verifies postal addresses for deliverability, classifies them as residential or commercial, and corrects formatting issues.
- [Documentation](https://developer.fedex.com/api/en-us/catalog/address-validation/v1/docs.html)

### FedEx Pickup API
Schedule, modify, and cancel package pickups, and determine pickup availability for a given origin and service combination.
- [Documentation](https://developer.fedex.com/api/en-us/catalog/pickup/v1/docs.html)

### FedEx Locations API
Find FedEx Office, Ship Centers, drop boxes, and authorized ship centers near a given address or coordinate.
- [Documentation](https://developer.fedex.com/api/en-us/catalog/locations/v1/docs.html)

### FedEx Authorization API
Issues OAuth 2.0 access tokens used to authenticate FedEx API calls.
- [Documentation](https://developer.fedex.com/api/en-us/catalog/authorization/v1/docs.html)

### FedEx Shipment Visibility Webhook
Pushes near real-time tracking events to a registered HTTPS endpoint, eliminating the need to repeatedly poll the Track API.
- [Documentation](https://developer.fedex.com/api/en-us/catalog/svm/v1/docs.html)
